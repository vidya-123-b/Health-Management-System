Hospital Management System

A web-based Hospital Management System built using PHP, MySQL, HTML, CSS, and JavaScript. The system streamlines hospital operations by managing patients, doctors, appointments, medical records, and administrative workflows. It includes secure, role-based dashboards for Admin, Doctors, and Patients.
